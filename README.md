# 主要方法

| Methods | Options |
| ---- | ----|
| rain.create() |  id(红包ID), amount(红包的金额) |
| rain.start()  |  data(ajax请求返回的数据) |
| rain.stop()   |  无需传参|
| rain.move()   |  rains（红包数组）|
| rain.clear()  |  无需传参 |
| rain.ajax()   |  类似jQuery中的$.ajax |
